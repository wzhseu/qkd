# README

## 小车B-发送端 前端

> 实现小车A与小车B通信
>
> 小车A与B将首先进行DAA认证，然后向各自的边缘网关发送获取量子密钥请求，获得通过无线信道密钥加密的量子密钥并解密至终端，然后进行量子加密通信
>
> 最终效果：
>
> 小车A收到小车B发送的视频文件
>
> 小车A和小车B进行视频通信

### 配置IP

打开config.ini文件，查看并修改相关IP地址：

local_ip：本机ip

to_ip：要进行通信的对端小车ip

cloud_ip：云服务器公网ip（已配置好，无需更改）

edge_ip：上层边缘网关ip

### 运行代码：

* backend/app: 进入终端命令

  ```
  flask run
  ```

* 从浏览器进入127.0.0.1页面，或直接ctrl+点击命令行内网址进入

* 等待对端小车A点击“实验开始”连接成功后，本端小车B点击“实验开始”

* 日志出现更新表明连接云服务器成功

### 前端如有修改：

  * frontend: npm run dev
  * frontend: npm run build
